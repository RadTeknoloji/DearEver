Política de Privacidad de DearEver

DearEver se compromete a proteger tu privacidad. Esta política describe cómo recopilamos, usamos y protegemos tu información.

Información que recopilamos

  • Información del perfil de usuario (nombre, correo electrónico)
  • Perfiles de memoriales que creas
  • Fotos y recuerdos que subes
  • Datos de ubicación (para encontrar memoriales cercanos)
  • Información del dispositivo

Cómo usamos tu información

  • Para proporcionar servicios de aplicación
  • Para sincronizar tus datos a través de CloudKit
  • Para enviar notificaciones
  • Para mejorar nuestros servicios

Almacenamiento de datos

Tus datos se almacenan de forma segura en CloudKit de Apple. No vendemos tu información personal.

Contacto

Para consultas de privacidad, contáctanos.
